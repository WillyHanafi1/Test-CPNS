# Materi TKP CPNS 2024: Tes Karakteristik Pribadi

Gunakan panduan ini untuk memilih jawaban dengan skor tertinggi (Poin 5) di modul TKP.

## 1. Strategi Utama: SJT (Situational Judgment Test)
- **Tujuan**: Mengukur keselarasan perilaku dengan profil Aparatur Sipil Negara (ASN) masa depan.
- **Trik Utama**: Jangan memilih jawaban yang sekadar "baik secara moral", pilih yang **"Paling Ideal Secara Birokrasi"** (Proaktif, Solutif, Profesional).

## 2. Indikator & Fokus Jawaban Skor 5
| Domain TKP | Fokus & Indikator Jawaban Terbaik (Poin 5) |
| :--- | :--- |
| **Pelayanan Publik** | Respons cepat, proaktif mencari solusi, fokus pada kepuasan pelanggan, anti-birokrasi kaku. |
| **Jejaring Kerja** | Kolaboratif, asertif, meredam ego, terbuka terhadap kritik fungsional, inklusif. |
| **Sosial Budaya** | Mudah berbaur (akulturasi), empati tinggi terhadap adat istiadat, anti-intoleransi, netral. |
| **Pemanfaatan TIK** | Menggunakan teknologi untuk efisiensi, autodidak belajar software baru, inisiatif digitalisasi. |
| **Profesionalisme** | Memisahkan urusan pribadi & kerja, prioritas tugas negara, tetap kerja prima saat krisis. |
| **Anti-Radikalisme** | Kritis memverifikasi hoaks/SARA, berani melapor sesuai regulasi, komitmen Pancasila mutlak. |

## 3. Resolusi Konflik (Conflict Matrix)
- **Problem**: Kolega tidak suka/merusak ekosistem.
- **Salah**: Konfrontasi emosional atau melarikan diri (mutasi).
- **Skor 5**: **Stoikisme Fungsional**. Terima fakta sosial, tetap fokus pada kinerja profesional untuk membuktikan nilai diri.

- **Problem**: Tekanan tenggat waktu & ketiadaan rekan.
- **Salah**: Meminta perpanjangan waktu atau mengeluh fasilitas.
- **Skor 5**: **Inisiatif Kepeloporan**. Ambil beban kerja ekstra untuk memastikan target tercapai tanpa keluhan.
